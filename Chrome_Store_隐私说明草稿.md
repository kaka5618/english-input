# Chrome 商店隐私说明草稿

## 数据使用说明

AI English Input 仅在用户主动触发生成时读取当前聚焦输入框中的文本内容，用于生成英文回复候选。插件不会自动读取聊天记录、邮件正文、网页历史内容或页面中的其他信息。

## 权限说明

### `storage`

用于在浏览器本地保存匿名用户 ID、默认场景、默认语气和今日剩余次数。数据仅保存在 `chrome.storage.local` 中。

### `activeTab`

用于在用户当前正在操作的页面中展示候选框，并将用户选择的英文候选写回当前输入框。

### `<all_urls>`

用于支持 WhatsApp Web、Gmail、Reddit 以及其他跨境电商/社媒网站中的输入框适配。插件只在用户主动按快捷键或连续空格触发时读取当前输入框内容，不会扫描整页内容。

## 后端处理说明

用户触发生成后，插件会将当前输入框文本、场景、语气和匿名用户 ID 发送到后端服务，用于调用 AI 模型生成英文回复候选和计算每日免费额度。后端不返回、不展示 API Key，插件侧也不保存任何 AI 服务密钥。

## 不收集的数据

- 不收集账号密码
- 不收集支付信息
- 不收集完整聊天记录
- 不收集完整邮件线程
- 不读取未聚焦的页面内容
- 不自动发送任何消息或邮件
