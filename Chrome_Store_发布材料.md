# Chrome 应用商店发布材料

## 插件名称

AI English Input

## 简短说明

专为跨境电商和外贸沟通设计：不是逐字翻译，而是把中文业务意图生成可直接回复客户的专业英文候选。

## 详细描述

AI English Input 是一款专为跨境电商客服、外贸销售和海外社媒运营设计的 Chrome 英文沟通助手。它和普通网页翻译工具不同，不只是把中文逐字翻成英文，而是理解你的业务意图，生成更适合客户沟通的英文回复候选。

你可以在 WhatsApp Web、Gmail、Reddit 以及常见网页输入框中直接用中文写下沟通意图，然后通过快捷键生成 3 条不同风格的英文回复候选。选择合适的候选后，插件会将英文内容写回当前输入框，由你自行检查并发送。

适合场景：

- 物流延迟解释
- 退款和售后沟通
- 催付款和订单确认
- 报价、询盘和外贸邮件
- Reddit 等海外社区互动
- 跨境电商客服日常回复

核心功能：

- 支持中文意图生成英文回复
- 一次生成 3 条候选表达
- 支持礼貌自然、简洁直接、正式专业、安抚道歉等语气
- 支持跨境客服、外贸邮件、社媒互动、通用翻译等场景
- 支持 WhatsApp Web、Gmail、Reddit 和常见 textarea/contenteditable 输入框
- 支持 `Ctrl + Shift + E` 快捷键触发
- 免费版每日 15 次，每次最多 100 字

隐私说明：

插件只会在用户主动触发生成时读取当前聚焦输入框中的文本，不会自动读取聊天记录、邮件线程或网页其他内容。生成结果只会写回当前输入框，不会自动发送消息或邮件。

## 权限说明

### `storage`

用于在浏览器本地保存匿名用户 ID、默认场景、默认语气和今日剩余次数。

### `activeTab`

用于在当前页面中展示候选框，并把用户选择的候选写回当前输入框。

### `<all_urls>`

用于适配 WhatsApp Web、Gmail、Reddit 以及其他跨境电商/社媒网站中的输入框。插件只在用户主动触发时读取当前聚焦输入框内容，不会扫描整页内容。

## 隐私政策填写要点

- 不收集账号密码
- 不收集支付信息
- 不读取完整聊天记录
- 不读取完整邮件线程
- 不自动发送任何消息或邮件
- 用户主动触发时，会将当前输入框文本发送到后端用于生成英文候选
- 后端使用匿名用户 ID 进行每日免费额度统计
- API Key 仅保存在后端环境变量中，插件侧不保存 AI 服务密钥
